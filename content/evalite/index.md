## Key Features

- **Type-safe** - Full TypeScript support with end-to-end type safety
- **Async-first** - Native async/await patterns for modern LLM calls
- **Lightweight** - Minimal dependencies, fast execution
- **Flexible scoring** - Compose custom scorers with simple functions
- **Test runner integration** - Works with Vitest, Jest, and other test frameworks
- **Local-first** - No external services required, run evals in your codebase
- **No vendor lock-in** - Open source and framework agnostic
