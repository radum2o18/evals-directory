## Key Features

- **Automatic tracing** - Zero-config tracing for all LangChain calls
- **Built-in evaluators** - Pre-built evaluators for RAG, QA, classification, and more
- **Dataset management** - Version-controlled test cases with collaborative editing
- **LangChain native** - First-class integration with LCEL, chains, and agents
- **Production monitoring** - Real-time observability with automatic logging
- **Playground** - Interactive prompt iteration and debugging
- **Human feedback** - Annotation workflows and feedback loops
